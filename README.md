# vpn_check

get vpn active status for Android, iOS, MacOS, Windows, Linux

## Getting Started

a simple usage example:

```
import 'package:vpn_check/vpn_check.dart';

bool isVpnActive = await VPNCheck.isVpnActive;
```
## Support Platform

- [x] Android
- [x] IOS
- [ ] MacOS
- [ ] Windows
- [ ] Linux
